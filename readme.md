# 코딩 테스트

## 1. 숫자 문자열과 영단어 - 2021 카카오 채용연계형 인턴십

### 문제 설명
네오와 프로도가 숫자놀이를 하고 있습니다. 네오가 프로도에게 숫자를 건넬 때 일부 자릿수를 영단어로 바꾼 카드를 건네주면 프로도는 원래 숫자를 찾는 게임입니다.

### 입출력 예
|s(입력)              | result |
|--------------------|--------|
| "one4seveneight"   | 1478   |
| "23four5six7"      | 234567 |
| "2three45sixseven" | 234567 |
| "123"              | 123    |

### 풀이 클래스
- kakaoCodingTest.EnglishWordWithDigit

### 풀이 과정에서 찾아 본 것
- map 사용법
- matches method 사용법
